# EcoEnjoy — Sitio estático

Este repositorio contiene la página estática del proyecto EcoEnjoy / Saborilla.

Cómo publicar en GitHub Pages (resumen):

1. Crear un repositorio en GitHub (web) o usar `gh repo create`.
2. Inicializar git localmente: `git init && git add . && git commit -m "Initial commit"`.
3. Añadir remoto `origin` y `git push -u origin main`.
4. En GitHub: `Settings -> Pages` y seleccionar `Branch: main` y `Folder: / (root)`.
5. Esperar unos segundos/minutos y abrir `https://<TU_USUARIO>.github.io/<REPO>/`.

Notas:
- `index.html` debe quedar en la raíz del repositorio (ya está en la raíz del proyecto).
- Si prefieres, puedo crear un flujo de GitHub Actions para publicar automáticamente.
